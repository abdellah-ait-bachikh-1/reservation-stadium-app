# Stadium Reservation System

A production-ready web application for managing stadium reservations, availability, and bookings.
Built with Next.js, React, and a modern Node.js stack.

---

## 🚀 Features

- Stadium availability and booking management
- User authentication (NextAuth)
- Internationalization (i18n)
- Real-time updates (Pusher)
- Email notifications
- Secure password handling
- Responsive UI with HeroUI & Tailwind CSS

---

## 🧰 Tech Stack

### Frontend
- Next.js 16
- React 19
- Tailwind CSS 4
- HeroUI
- Framer Motion

### Backend
- Node.js 18+
- Next.js API Routes
- NextAuth (authentication)
- Drizzle ORM
- MySQL

### Utilities
- Zod (validation)
- Nodemailer (emails)
- Pusher (real-time)
- next-intl (internationalization)

---

## 📋 Requirements

- Node.js **18.x or later** (recommended via `.nvmrc`)
- npm
- MySQL database
